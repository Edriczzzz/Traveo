Panel de Administración

Bienvenido al panel de administración. Aquí puedes gestionar las diferentes secciones del sistema:

  • Gestión de Usuarios: Editar o eliminar usuarios, asignar roles y gestionar permisos.
  • Gestión de Vuelos: Editar o eliminar Vuelos, asignar roles y gestionar permisos.
  • Gestión de Aerolineas: Crear, Editar o eliminar Aerolineas, asignar roles y gestionar permisos.
  • Gestión de Ciudaddes: Crear, editar o eliminar ciudades, asignar roles y gestionar permisos.
